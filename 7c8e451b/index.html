日语变形

2020-04-16 分类 日语 作者 ~1.56K 字
本文最后编辑于 前，其中的内容可能需要更新。

动词变形

动词ます形（連用形）

  • 一类动词：将结尾う段假名变成い段假名后+ます
  • 二类动词：去る＋ます
  • 三类动词：来（く）る→来（き）ます する→します

动词ない形(未然形)

  • 一类动词：将结尾う段假名变成あ段假名后+ない。
  • 二类动词：去る+ない。
  • 三类动词：来（く）る→来(こ)ない する→しない。

动词て形（連用形）

  • 一类动词：
    い音变：く→いて　ぐ→いで
    促音变：う、つ、る为结尾的,去掉う、つ、る＋って
    拨音变：む、ぬ、ぶ为结尾的去掉+む、ぬ、ぶ＋んで
    以す为结尾的：す→して
  • （2）二类动词：去る+て
  • （3）三类动词：来る→来（き）て する→して

动词た形（連用形）

  • 一类动词：
    い音变：く→いた　ぐ→いだ
    促音变：う、つ、る为结尾的,去掉う、つ、る＋った
    拨音变：む、ぬ、ぶ为结尾的去掉+む、ぬ、ぶ＋んだ
    以す为结尾的：す→した
  • 二类动词：去る+た
  • 三类动词：来る→来（き）た する→した

动词命令形

  • 一类动词：将结尾う段假名变成え段假名即可.
  • 二类动词：去る+ろ
  • 三类动词：来る→来（こ）い する→しろ

动词可能形

  • 一类动词：将结尾う段假名变成え段假名后+る
  • 二类动词：去る+られる
  • 三类动词：来る→来（こ）られる する→できる

动词假定形

  • 一类动词：将结尾う段假名变成え段假名后+ば
  • 二类动词：去る+れば
  • 三类动词：来る→来（く）れば する→すれば
  • 形容词：去い+ければ

例：安い→安ければ　　おいしい→おいしければ

  • 形容动词：去だ+なら

例：暇だ→暇なら　　　簡単だ→簡単なら

  • 名词：名词直接+なら

例：日本人→日本人なら

动词意向形

  • 一类动词：将结尾う段假名变成お段假名+う
  • 二类动词：去る+よう
  • 三类动词：来る→来（こ）よう する→しよう

动词被动形

  • 一类动词：将结尾う段假名变成あ段上假名后+れる
  • 二类动词：去る+られる
  • 三类动词：来る→来（こ）られる する→される

动词使役形

  • 一类动词：将结尾う段假名变成あ段上假名后+せる
  • 二类动词：去る+させる
  • 三类动词：来る→来（こ）させる する→させる

动词使役+被动形

  • 一类动词：将结尾う段假名变成あ段假名后+される
  • 二类动词：去る+させられる
  • 三类动词：来る→来（こ）させられる する→させられる

动词禁止形

  • 原形、基本形+な

形容词变形

常规变形

  1. た形，表过去式，对应的敬体为：でした
    な形容词/名词：词尾+だった（好き ->　好きだった）
    い形容词：去掉词尾的い + かった（弱い ->　弱かった）
  2. ない形，表否定式，对应的敬体为：ではありません
    な形容词/名词：词尾+でわない，ありません的简体为ない。
    い形容词：词尾的い -> く + ない（悪い ->　悪くない）
  3. 过去否定式，对应的敬体为：ではありませんでした
    な形容词/名词：词尾+でわなかった，因为ありませんでした的简体为なかった。
    い形容词：去掉ない形的词尾い + かった（若い ->　若くなかった）
  4. て形，用于多个形容词同时进行修饰的场景
    な形容词/名词：直接 + で（綺麗で、静かな町）
    い形容词：去掉词尾的い -> く + て（高くて、大きい山）
  5. 后面接动词(例：成为/变成…：+なります)
    な形容词/名词：直接 + に（医やになります）
    い形容词：去掉词尾的い -> く（明るくなります）
  6. な形容词：词尾+な+名词；词尾+に+动词；词尾+だ用于结句（语气较强，女生一般不用）。
    名词：词尾+の+名词；词尾+に+动词；词尾+だ用于结句（语气较强，女生一般不用）。
    い形容词：直接+名词；去掉词尾的い+く+动词；可直接结句；去掉词尾い+さ，则形容词变名词。
